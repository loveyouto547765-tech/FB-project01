วิธีขอ Page Access Token
กรอง:
⚠️ วิธีแก้ Error "pages_read_engagement"
  1. ไปที่ Graph API Explorer
  2. เลือก App ของคุณด้านบนขวา
  3. คลิก "Generate Access Token" (ปุ่มสีน้ำเงิน)
  4. ติ๊ก Permission ให้ครบทั้งหมด:
    pages_show_list
    pages_read_engagement
    pages_manage_posts
    pages_read_user_content
  5. คลิก "Generate Token" → อนุมัติใน popup
  6. คัดลอก Token ใหม่มาใส่แทนอันเก่า แล้วกด ดึงข้อมูลเพจ
ถ้ายังไม่ได้ — ตรวจสอบ App Mode
  1. ไปที่ My Apps → เลือก App
  2. เมนูซ้าย → Settings → Basic
  3. ดูว่า App Mode เป็น Development หรือ Live
  4. ถ้าเป็น Development: ใช้ได้เฉพาะ Facebook Account ที่เป็น Admin ของ App เท่านั้น
  5. เพิ่ม Facebook Account ของคุณเป็น Tester ได้ที่: Roles → Test Users

💡 สำหรับใช้งานส่วนตัว ไม่ต้อง Submit for Review — แค่ใช้ Account ที่เป็น Admin ของ App ก็พอ
